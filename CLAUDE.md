# Seisly

SEIS and EIS advance assurance platform for UK startups.

## What we are building
A web app that makes applying for SEIS and EIS advance assurance as easy and cheap as possible. Competing directly with SeedLegals and undercutting them on price.

## Tagline
Need SEIS or EIS advance assurance? Seisly done.

## Pricing
- SEIS only: £179 one-time
- EIS only: £179 one-time
- SEIS and EIS: £249 one-time
- Express Review add-on: £100
- Resubmission: £50
- No subscriptions ever

## Tech stack
- Next.js 15 with App Router
- TypeScript
- Tailwind CSS
- Supabase (Postgres, Auth, Storage)
- Anthropic API (Claude) for AI document generation
- Stripe for payments
- Resend for email
- React-PDF for document generation

## Key features to build
1. Eligibility checker (free, no account needed)
2. Multi-step application flow
3. AI-generated advance assurance covering letter
4. SEIS1/EIS1 form completion
5. Investor certificate generation (SEIS3/EIS3)
6. HMRC query support
7. Compliance reminders (3-year window)
